# Asymmetric architecture and nest site structural properties


This is a repository that contains all code and data needed to reproduce the results of an article investigating the relationship between nest asymmetric architecture and pre-existing structures in the nest site.


The repository consists of an Rproject with 1 script and 2 folders containing the data  needed to run these scripts, including the raw figures. For a detailed description of all the files, please see below. To run this repository, first download and unzip the entire repository and open the Rproject file: ProyectoVickyAsymm.Rproj.
